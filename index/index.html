Первое задание

Сделать обёртку над массивами, которая будет иметь вид const arr = useArray([]) и будет обладать такими методами
(Эти методы работают со СТАРЫМ массивом):

push - похоже на нативный метод, но не возвращает length
unshift - похоже на нативный метод, но не возвращает length
pop - похоже на нативный метод, но не возвращает element
shift - похоже на нативный метод, но не возвращает element
clear - очистка массива
removeIndex - удаление элемента массива по его индексу (со смещением остальных элементов)
move - перемещает элемент с одного индекса на другой, двигая остальные элементы

Эти методы возвращают НОВЫЙ массив, не изменяя старый (называется имьютабельность (immutable)):

removeById - если массив состоит из ОБЪЕКТОВ со специфичными id-полями, то удалять объекты по этому полю, но имейте ввиду, что метод должен уметь принимать на удаление как массив id-шников, так и просто id. Также второй аргумент функции - это string, который указывает на то, какое название имеет id-поле (например в объекте нет поля "id", но есть поле "key", которое также указывает на уникальность значения), по-дефолту аргумент должен быть равен "id"
modifyById - тоже самое, что и предыдущий метод, но только ЗАМЕНЯЕТ, а не удаляет.

Пример использования: arr.modifyById(2, {...newObjData}, "key").
